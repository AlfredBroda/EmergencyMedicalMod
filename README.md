# Emergency Medical Mod
This is a modification for the game [Ostranauts](https://steamcommunity.com/app/1022980), by Blue Bottle Games.

Adds Medical items such as an Injector Tool with various drug capsules, such as:
 - WakeUp - for rousing an unconscious patient
 - SedateU - for sedating rowdy patients
 - QuickHeal - for speeding up health recovery

These will appear in The OKLG Medical Kiosk if [FFU: BR](https://git.soon-tm.info/warstalker/FFU-BR#precision-array-modifications) patches are available.
